# 2025年12月11日 - 火箭模拟之旅 Day 1

## 今日目标
- [x] 安装并初步熟悉 OpenRocket 软件。
- [ ] 理解“重心 (CG)”和“压力中心 (CP)”的关系。

## 我学到了什么 (What I Learned)
1.  **OpenRocket 安装**: 过程很顺利，软件是基于 Java 的，跨平台性很好。
2.  **基本操作**: 通过拖拽组件（头锥、箭体、尾翼）可以快速搭建一个模型。每个组件的参数（长度、直径、材料）都可以修改。
3.  **重心的初步理解**: 软件会自动计算重心位置，它基本上是所有组件质量的加权平均点。

## 遇到的问题 (Challenges)
*   一开始不明白为什么我的火箭稳定性 (Stability) 是负数。
*   软件里的发动机列表非常多，不知道该如何选择。

## 解决方案 / 待办事项 (Solutions / TODO)
*   **问题1的解决方案**: 查阅资料后发现，**压力中心必须在重心之后**，火箭才能稳定。我通过增大尾翼面积或在头部增加配重，成功让稳定性变为正值。
*   **问题2的待办**: 下一步需要研究一下模型火箭发动机的参数，比如总冲、推力曲线等代表什么意思。
